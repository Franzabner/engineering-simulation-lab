# Field Propulsion Hypothesis Taxonomy

Status: scaffolded
Publication state: public scaffold
Release state: not released

## Purpose

This taxonomy supports public-safe speculative engineering notes by separating hypothesis framing from evidence, validation, or deployment claims.

## Taxonomy

| Term | Public-safe meaning |
| --- | --- |
| Concept category | A neutral label for an idea under review. |
| Hypothesis | A question to evaluate, not a result. |
| Assumption | A synthetic or reviewed public-safe premise. |
| Limitation | A constraint that blocks stronger claims. |
| Review gate | Human review required before any result, benchmark, or proof language. |

## No-Result Posture

This scaffold contains no measured results, benchmark outputs, raw traces, validated simulation, physical validation, deployment claim, production-readiness claim, or proof-completion claim.

## Held Out

Do not include breakthrough claims, anomaly conclusions, proprietary methods, private traces, customer claims, deployment claims, production claims, or any wording that treats a hypothesis as demonstrated.
