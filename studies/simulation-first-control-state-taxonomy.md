# Simulation-First Control State Taxonomy

Status: scaffolded
Publication state: public scaffold
Release state: not released

## Purpose

This taxonomy describes control-state labels for simulation-first study notes. It is not live control logic and does not represent a deployed system.

## Control-State Fields

| Field | Public-safe rule |
| --- | --- |
| State name | Use a generic synthetic label. |
| State purpose | Describe study intent only. |
| Entry condition | Use placeholder conditions, not real telemetry or private measurements. |
| Exit condition | Use placeholder conditions, not production logic. |
| Review status | Use planned, scaffolded, public scaffold, or not released. |

## Held Out

Do not include production control logic, private topology, private source paths, endpoints, real production logs, energy output values, proven-output claims, deployment claims, safety approval claims, certification claims, benchmark results, production-readiness claims, or proof-completion claims.

## Use Rule

Use these labels to organize public study documentation only. Any real implementation detail requires separate private review.
